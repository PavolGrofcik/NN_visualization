# NN_visualization

Názorná animácia regresnej 3 vrstvovej NN siete
## NN parametre
Vrstvy: 3  
Vstupy: 3  
Aktivačné funkcie:  
  Skrytá vrstva: ReLu  
  Výstupná vrstva: Identita = 1  
  
 
Dataset: **QSAR fish toxicity Data Set**  
Link: https://archive.ics.uci.edu/ml/datasets/QSAR+fish+toxicity
